Spring Boot auto-configuration for Camel OpenTelemetry (legacy).

This starter integrates Camel with https://opentelemetry.io[OpenTelemetry] to provide distributed tracing for Camel routes. It auto-configures an `OpenTelemetryTracer` that creates spans for each Camel processor and propagates trace context across routes and components.

NOTE: This is the legacy OpenTelemetry integration. Consider using `camel-opentelemetry2-starter` instead, which is the newer implementation.

Use the `@CamelOpenTelemetry` annotation on your Spring Boot main class or configuration to enable.
